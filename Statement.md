Managing physical books in libraries, schools, colleges, and personal collections is often done manually. This leads to several issues:

Difficulty in tracking which books are available or rented

No proper record of books returned or overdue

Human errors in maintaining registers

Time-consuming book search process

These problems make the process inefficient, especially when the number of books increases.

The purpose of this project is to build a simple, efficient, and user-friendly Book Rental System that:

Provides a clean graphical interface using Tkinter

Saves data automatically using a JSON file


This project eliminates manual record-keeping and makes book management quick, accurate, and organized.
